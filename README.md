# QA First Steps

Repositório criado para praticar conceitos de Quality Assurance (QA),
incluindo testes manuais, cenários de teste e introdução à automação.

## Objetivo
Retomar a prática com QA e desenvolvimento, criando uma base sólida
para atuar como QA Júnior.

## Conteúdo
- Casos de teste manuais
- Cenários positivos e negativos
- Exemplos de bug report
- Introdução à automação de testes
